# Personal Resume Website

A responsive personal resume website showcasing my profile, work experience, education, and skills. This project serves as a digital CV to highlight my academic background and internships.

## Live Demo

[View the Website](https://vp1107.github.io/)

## Features

- **Responsive Design**: Optimized for both desktop and mobile viewing with a clean, user-friendly interface.
- **Structured Layout**:
  - **Sidebar**: Quick access to contact info, skills, and social links.
  - **Main Content**: Detailed sections for Profile, Work Summary, and Education.
- **Modern Typography**: Utilizes 'Roboto' from Google Fonts for better readability.

## Technologies Used

- **HTML5**: Used for the semantic structure of the webpages.
- **CSS3**: Used for styling, layout, and responsive design.

## Project Structure

```
resume-website/
├── assets/          # Images and other static assets
├── css/
│   └── style.css    # Main stylesheet
├── index.html       # Main HTML file
└── README.md        # Project documentation
```

## Getting Started

To run this project locally on your machine:

1. **Clone the repository**:
   ```bash
   git clone https://github.com/VP1107/vp1107.github.io.git
   ```

2. **Navigate to the project directory**:
   ```bash
   cd vp1107.github.io
   ```

3. **Open the website**:
   - Locate `index.html` in the folder.
   - Double-click to open it in your default web browser.
